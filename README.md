# Crypto Trading API v2

Advanced cryptocurrency trading signals API with real-time price data and technical analysis.

## Features

- Real-time price data from multiple exchanges (Binance, CoinGecko, KuCoin)
- Technical analysis signals (RSI, MACD, Bollinger Bands)
- Support for multiple timeframes (1m, 5m, 15m, 1h, 4h)
- Scalp trading signals
- Market status monitoring
- RESTful API with FastAPI
- CORS enabled for web applications
